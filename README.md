# 🎬 IMDB Movie Analysis Dashboard

This project analyzes the *Top 1000 movies dataset* from IMDb using data visualization.  
The goal of this project is to identify audience preferences, genre performance, director contributions, and movie release trends.

##  Key Analysis
- Genre-wise rating analysis
- Director-wise movie count and gross comparison
- Top 10 highest grossing movies
- Runtime vs gross comparison
- Movie release trend over time (1990–2015)

##  Key Insights
- Western genre received the highest average rating (8.8).
- Alfred Hitchcock directed the highest number of movies.
- Steven Spielberg generated the highest gross income.
- The movie *1917* has the highest gross revenue, while *3 Idiots* has a longer runtime but lower gross.
- Movie releases increased significantly between 1990 and 2015.

##  Tools Used
- Tableau (Data Visualization)
- IMDb Dataset

##  Objective
To transform raw movie data into meaningful insights using visual analytics.
